# fullstack login template

## client

```
yarn install
```

```
yarn start
```


## server

```
npm install
```
```
npm run dev
```

### server contains .env file with this values
```
SERVER_PORT
MONGO_URL
SECRET_KEY_JWT
AWS_AVATARS_BUCKET
AWS_BUCKET_REGION
AWS_ACCESS_KEY
AWS_SECRET_ACCESS_KEY
```
